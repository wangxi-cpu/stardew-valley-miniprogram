# 星露谷物语 - 微信小游戏

一款完整的像素风格农场经营游戏，灵感来源于星露谷物语。

## 游戏特色

### 🌾 农场系统
- 24种作物，四季种植
- 作物品质分级（普通、银星、金星、铱星）
- 畜牧系统（奶牛、山羊、鸡、猪等）
- 建筑系统（畜棚、鸡舍、温室等）

### 👥 社交系统
- 12位可结婚对象
- 30+普通村民
- 送礼、对话、结婚
- 好感度和心事件

### ⛏️ 探索系统
- 120层矿洞
- 挖矿、战斗
- 怪物和战利品

### 🎣 钓鱼系统
- 30+种鱼类
- 不同季节、天气、地点
- 钓鱼小游戏

### 🎨 制作系统
- 50+种制作配方
- 工具升级
- 烹饪和加工

### 📖 剧情任务
- 主线任务
- 支线任务
- 成就系统

## 技术架构

- 微信小游戏框架
- Canvas 2D渲染
- 模块化设计
- 本地存档

## 文件结构

```
stardew-complete/
├── game.js              # 游戏入口
├── game.json            # 游戏配置
├── app.js               # 主应用
├── app.json             # 应用配置
├── app.wxss             # 全局样式
├── pages/               # 页面
├── utils/               # 工具模块
├── data/                # 游戏数据
└── assets/              # 资源文件
```

## 运行方式

1. 下载微信开发者工具
2. 导入项目目录
3. 使用测试号或正式AppID
4. 编译运行
